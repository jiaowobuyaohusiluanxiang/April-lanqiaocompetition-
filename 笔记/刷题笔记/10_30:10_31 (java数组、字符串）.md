####        2.二维数组打印

​             2.1 for循环打印

```java
public class test01 {
    public static void main(String[] args) {
        int [][] a=new int[5][6];
        for(int i=0;i<a.length;i++){
            for(int j=0;j<a[i].length;j++){
                System.out.println(a[i][j]);
            }
        }
    }
}

```
